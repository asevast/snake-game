# 🐍 Snake Game

Enhanced Snake Game with modern features built with vanilla JavaScript, HTML5 Canvas, and CSS.

## Features

- 🎮 **Gameplay**: Speed progression, 3 difficulty levels, high score tracking
- ⏸️ **Controls**: Keyboard (Arrow keys/WASD), swipe gestures, on-screen D-pad
- 🎨 **Visuals**: Gradients, shadows, animated food, snake eyes, particle effects
- 🌗 **Themes**: Dark/Light mode toggle with persistence
- 🔊 **Audio**: Sound effects for eating, milestones, and game over
- 📱 **Mobile**: Fully responsive design with touch controls
- ♿ **Accessible**: ARIA labels and keyboard navigation

## Play Now

Open `index.html` in your browser or visit the live demo.

### Controls

| Action | Keyboard | Mobile |
|--------|----------|--------|
| Move | Arrow Keys / WASD | Swipe / D-pad |
| Pause | Space | Pause button |
| Start | Enter | Start button |
